# hello_world_web

A new Flutter project for web.

## Requirements

- Flutter: 3.38.0
- Android SDK: 36
- Android BuildTools: 28.0.3
- XCode: 26.1
- Chrome Browser

## Getting Started

1. Run the following command to make sure web development in enabled

```
fvm flutter config --enable-web
```

2. Verify Flutter recognize web browser

```
>> fvm flutter devices
Found 1 connected devices:
  Chrome (web)    • chrome • web-javascript • Google Chrome 143.0.7499.193
```

3. Fetch all dependencies

```
fvm flutter pub get
```
